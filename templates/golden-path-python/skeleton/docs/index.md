# Golden Path – Python Service

This documentation describes the standard Golden Path service generated by the Internal Developer Platform.

## Overview

This service represents a standardized starting point for backend services within the organization.

It is designed to be:
- Easy to understand
- Kubernetes-ready
- Consistent across teams

## Architecture

- Language: Python
- Framework: FastAPI
- Runtime: Docker
- Deployment target: Kubernetes (AKS)
- Namespace: `idpapp`

## How to use this service

1. Extend the Python application in `app/main.py`
2. Build the container image using the provided Dockerfile
3. Deploy to Kubernetes using the manifests in `k8s/`
4. Register the service in Backstage using `catalog-info.yaml`

## Ownership and Governance

Ownership is defined via Backstage using logical identifiers (e.g. groups from the identity provider).

Roles and groups can be integrated later via OIDC (Keycloak) without changing the service.
